#Automação de Geração de IDs Únicos com Google Apps Script

## 📌 Sobre o projeto

Este projeto automatiza a geração de identificadores únicos (IDs) em planilhas no Google Sheets, garantindo padronização, rastreabilidade e confiabilidade dos dados.

A solução foi desenvolvida para cenários operacionais onde há necessidade de controle por linha, eliminando processos manuais e reduzindo erros.

---

## ⚙️ Tecnologias utilizadas

* Google Apps Script
* Google Sheets

---

## 🚀 Funcionalidades

* Geração automática de IDs únicos por linha
* Prevenção de duplicidade de identificadores
* Execução em lote para melhor performance
* Integração com menu personalizado na planilha

---

## 🧠 Problema resolvido

A ausência de identificadores únicos em planilhas pode causar:

* Duplicidade de registros;
* Falta de rastreabilidade;
* Erros operacionais;
* Dificuldade em integrações futuras;

O script resolve esses problemas ao gerar IDs automaticamente apenas quando necessário, garantindo consistência nos dados.

---

## ▶️ Como usar

### 1. Configurar o script

* Acesse o editor de Apps Script
* Cole o código do arquivo (nome do seu script).gs

### 2. Ajustar parâmetros

No código, configure conforme sua planilha:

* Nome da aba (ABA)
* Linha inicial (LINHA_INICIAL)
* Coluna de verificação (ex: coluna A) 
* Coluna onde será gerado o ID

### 3. Executar a automação

* Atualize a planilha
* Utilize o menu personalizado:
* Status → Gerar UUID
